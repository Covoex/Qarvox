# Qarvox
[![Build Status](https://travis-ci.org/Covoex/Qarvox.svg?branch=master)](https://travis-ci.org/Covoex/Qarvox)

## Installation
```
git clone https://github.com/Covoex/Qarvox
```

```
./gradlew run
```
